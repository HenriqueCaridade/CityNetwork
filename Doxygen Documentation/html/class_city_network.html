Detailed Description

Represents a city network with nodes and edges.

The CityNetwork class manages a city network consisting of nodes and edges. It provides various operations and algorithms for working with the city network.

Perform the backtracking algorithm to find the shortest path in the city network.

Returns
The shortest path found by the backtracking algorithm.

The time complexity of the backtracking algorithm is O(2^N), where N is the number of nodes in the city network.
